Iniciativa Open Source da Efforia

Nós acreditamos que para qualquer atividade é melhor executada quando coletivamente. Por isso a Efforia é incentivadora e contribuidora de projetos de Software Livre, mantendo um perfil próprio no GitHub e divulgando os principais projetos de software livre existentes para seus membros. A iniciativa torna-se completa com a existência da Efforia Labs, promovendo hackathons e dojos de forma a fomentar o código aberto.
